# WhereNow Use Case 說明

系統名稱：WhereNow－地點清單與情侶回憶管理系統

---

# 系統角色（Actors）

### User（一般使用者）
一般註冊會員，可以使用基本功能。

### Couple User（情侶使用者）
已建立情侶關係的使用者，可以使用共享功能。

### Admin（管理員）
負責管理系統資料與使用者。

---

# 核心 Use Cases

## 1. 使用者註冊與登入

### 說明
新使用者可以建立帳號並登入系統。

### 基本流程
1. 使用者進入註冊頁面
2. 輸入帳號與密碼
3. 系統建立帳號
4. 使用者登入系統

---

## 2. 情侶邀請與綁定

### 說明
使用者可以邀請另一位使用者建立情侶關係。

### 基本流程
1. 使用者發送情侶邀請
2. 對方收到邀請
3. 對方接受邀請
4. 系統建立情侶關係

---

## 3. 新增與管理地點清單

### 說明
使用者可以建立自己的地點清單。

### 功能包含
- 新增地點
- 編輯地點
- 刪除地點
- 設定分類與標籤
- 標記已去過

---

## 4. 隨機抽選地點

### 說明
系統可以從地點清單中隨機抽選一個地點。

### 基本流程
1. 使用者設定抽選條件
2. 系統隨機選出地點
3. 顯示抽選結果

情侶使用者可以進行 **共同抽選地點**。

---

## 5. 建立去過紀錄

### 說明
使用者可以記錄已去過的地點並留下回憶。

### 功能包含
- 上傳照片
- 填寫心得
- 給予評分
- 記錄花費
