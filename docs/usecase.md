# Use Case 說明

本系統名稱：WhereNow－地點清單與情侶回憶管理系統

---

# 主要角色 (Actors)

1. 一般使用者 (User)
2. 情侶使用者 (Couple User)
3. 系統管理員 (Admin)

---

# Use Case 1：使用者註冊

### 說明
新使用者建立帳號以使用系統。

### 基本流程
1. 使用者進入註冊頁面
2. 輸入帳號與密碼
3. 系統檢查資料是否正確
4. 建立新帳號
5. 註冊成功

---

# Use Case 2：使用者登入

### 說明
使用者登入系統以使用各項功能。

### 基本流程
1. 使用者進入登入頁面
2. 輸入帳號與密碼
3. 系統驗證帳號資料
4. 登入成功並進入系統首頁

---

# Use Case 3：建立情侶關係

### 說明
使用者可以邀請另一位使用者建立情侶關係。

### 基本流程
1. 使用者發送情侶邀請
2. 對方收到邀請通知
3. 對方接受邀請
4. 系統建立情侶關係

---

# Use Case 4：新增地點

### 說明
使用者可以新增想去或去過的地點。

### 基本流程
1. 使用者進入新增地點頁面
2. 輸入地點資訊
3. 系統儲存地點資料
4. 地點新增成功

---

# Use Case 5：新增回憶

### 說明
使用者可以在地點新增回憶與照片。

### 基本流程
1. 使用者選擇一個地點
2. 輸入回憶內容
3. 上傳照片
4. 系統儲存回憶資料
5. 回憶新增成功
